Este script foi desenvolvido exclusivamente para fins educacionais e de teste de segurança em ambiente controlado. Ele demonstra a técnica de registro de eventos de teclado (keypress monitoring) em Python.

O objetivo é mostrar como a entrada de dados pelo teclado pode ser interceptada para fins de estudo de contramedidas e proteção de endpoints. O log gerado é tratado internamente para fins de demonstração.

⚠️ Aviso: O uso desta técnica para monitorar ou coletar dados sem consentimento explícito e em máquinas de terceiros é ilegal.
